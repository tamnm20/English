# Vocabulary

## Cấu trúc thường gặp

look forward to
take a break
be kind to
Leave for the trip
tax return
track progress

Trật từ tính từ: O (opinion) - S (size) - A (age) - S (shape) - C (color) - O (origin) - M (material) - P (purpose)

Khi miêu tả đặc điểm, tính chất của một việc gì đó, ta có cấu trúc “It + to be + adj + to V” (làm gì đó thì như thế nào)
